# 機能概要

## Why

Claude Code CLI は強力だがターミナル操作が必要で、複数セッションの同時管理が難しい。
swift-agent-sdk を活用した GUI アプリにより、直感的な操作でマルチセッション管理を実現する。

## What

macOS ネイティブの SwiftUI アプリ。サイドバーでセッションを管理し、
チャット UI でメッセージを送受信する。ストリーミング表示とツール可視化により、
Claude の動作をリアルタイムで確認できる。

## Intent

- **SDK のリファレンス実装**: AgentClient / AgentSession / AgentMessage の全主要パスを網羅
- **マルチセッション**: 複数の独立した会話を並行管理
- **リアルタイム体験**: ストリーミングによる逐次テキスト表示
- **透明性**: ツール使用の可視化で Claude の動作を把握
- **軽量**: 最小限の機能で最大限の実用性

## FF 一覧表

| FF-ID | 名称 | 概要 | 画面数 | 関連要件 |
|-------|------|------|--------|----------|
| FF-001 | セッション管理 | セッションの作成・一覧表示・切替・再開・終了・削除 | 2 | FR-001〜FR-007 |
| FF-002 | チャットメッセージング | メッセージ送信・ストリーミング表示・応答表示 | 1 | FR-008〜FR-012 |
| FF-003 | ツール可視化 | ツール使用/結果のカード表示・折りたたみ | 0* | FR-013〜FR-015 |
| FF-004 | モデル・設定制御 | モデル選択・作業ディレクトリ・コスト/トークン表示 | 1 | FR-016〜FR-020 |
| FF-005 | データ永続化 | セッション情報・メッセージ履歴のローカル保存と復元 | 0* | FR-021〜FR-024 |

> *FF-003, FF-005 は独立画面を持たず、他の FF の画面内で動作する。
